# X9C-DigitalPot
X9C_DigiPot.cpp - Library for controlling the X9C102 1kohm,<br/>

                               X9C103 10kohm,<br/>
                               X9C104 50kohm,<br/>
                               X9C503 100kohm.<br/>
See the datasheet for PinConfig at https://www.intersil.com/content/dam/intersil/documents/x9c1/x9c102-103-104-503.pdf<br/>
