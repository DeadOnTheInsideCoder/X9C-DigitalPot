# X9C-DigitalPot
Library for controlling: 

                               X9C102 1kohm,
                               X9C103 10kohm,
                               X9C104 50kohm,
                               X9C503 100kohm.
See the datasheet for PinConfig at https://www.intersil.com/content/dam/intersil/documents/x9c1/x9c102-103-104-503.pdf<br/>
